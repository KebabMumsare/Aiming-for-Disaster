# Aiming for Disaster Unity Project

## Overview
Aiming for Disaster is a Unity project that features a simple currency management system. The project includes a main gameplay scene and is structured to facilitate easy management of player currency.

## Project Structure
```
Aiming-for-Disaster-Unity
├── Assets
│   ├── Scripts
│   │   └── Currency.cs
│   └── Scenes
│       └── Main.unity
├── Packages
│   └── manifest.json
├── ProjectSettings
│   └── ProjectSettings.asset
├── .gitignore
└── README.md
```

## Currency Management
The `Currency.cs` script manages the player's currency balance. It allows for:
- Adding currency
- Spending currency
- Setting the balance
- Notifying listeners of balance changes
- Updating UI elements with the current balance

## Setup Instructions
1. Open the project in Unity.
2. Ensure all necessary packages are installed as specified in `Packages/manifest.json`.
3. Open the `Main.unity` scene to start the gameplay.
